# Django-based-project

This project is based on django that helps translate sentences for a given english wikipedia page to any Indian Language.
This project has 3 pages. 

Dashboard.html: It displays Wikipedia Article Title and Target Language of all the entries in the Project Table. 

<img width="1440" alt="image" src="https://user-images.githubusercontent.com/52610520/228619748-28dff68b-3f63-447a-b3f6-3d358f4cdb4d.png">

Home.html: It allows user to add new entry into the project table. Once you click on submit, you will be redirected to the sentence page.

<img width="1440" alt="image" src="https://user-images.githubusercontent.com/52610520/228620251-42acf939-bf6a-4f89-b8b2-1eef8d2dec96.png">


Sentence.html: This page displays all sentences of the Wikipedia Article submitted in home page. It also gives the user option to add the translation for each sentence and to save into the Sentence Table.

<img width="1440" alt="image" src="https://user-images.githubusercontent.com/52610520/228620966-4f122973-66a1-415e-8318-2bf970fbcba1.png">

To Run this project, install the requirements.txt file which contains all the required packages.
